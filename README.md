# Proyecto Web - Centro MAC Áncash

## 🎯 Objetivo del Proyecto
Diseñar, estructurar y desarrollar la página web oficial e informativa para el Centro MAC Áncash (ubicado en Megaplaza Chimbote), implementando un sistema de tramitología digital y aplicando buenas prácticas de desarrollo frontend, accesibilidad universal y control de versiones colaborativo.

## 🛠️ Tecnologías que se usaron
* **HTML5:** Estructura semántica (BEM) para la correcta jerarquización de la información y SEO institucional.
* **CSS3:** Estilos y Diseño Responsive (Mobile First), uso de variables nativas para Modo Claro/Oscuro y animaciones fluidas.
* **JavaScript (ES6+):** Interactividad avanzada, enrutador de vistas (SPA), lógica del asistente virtual y consumo de librerías.
* **Integraciones Adicionales:** Firebase (Persistencia de datos para reservas), Chart.js (Gráficos del Dashboard) y QRCode.js (Generación de tickets).
* **Git & GitHub Flow:** Control de versiones, gestión de ramas y trabajo colaborativo en equipo.

## 🚀 Características Principales del Desarrollo
* **Arquitectura SPA (Single Page Application):** Navegación fluida e instantánea entre los módulos (Servicios, Pasaporte, Noticias, Reservas) sin necesidad de recargar la página web.
* **Diseño 100% Responsivo:** Interfaz gráfica que se adapta dinámicamente a celulares, tablets y computadoras de escritorio.
* **Accesibilidad Universal (Inclusión):** Integración de un panel con modo de "Alto Contraste" y "Aumento de Texto" para facilitar la navegación a adultos mayores o personas con dificultades visuales.
* **Asistente Virtual (Chatbot):** Sistema interactivo basado en un árbol de decisiones para orientar al ciudadano sobre requisitos, módulos y citas de entidades como RENIEC, Migraciones y Banco de la Nación.
* **Dashboard Analítico:** Panel de control de atención ciudadana con renderizado de gráficos estadísticos interactivos.
* **Sistema de Reservas Virtuales:** Flujo por pasos para agendar turnos, con validación de horarios y generación automática de un ticket con código QR listo para imprimir.
